What this ships first

The first version is intentionally small: docs, prompts, and a simple UI mockup. It is designed to become public proof-of-work before expanding into a testnet-connected prototype.

3

Payment-intent demo

A safe first agentic commerce flow: an agent prepares a USDC request, while the human keeps approval control.

Why this fits Arc

Arc's public narrative points toward stablecoin-native finance, agentic economy apps, autonomous agents, onchain identity, and predictable payment infrastructure. This kit turns that narrative into builder-facing workflows.
